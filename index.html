Objetivo del curso

Diseñar en forma de maquetación el Index o la interfaz de una página web utilizando como temática métodos de ordenación con programación orientada a objetos (Lógica de Programación).

Temáticas

Objetivo específico 1: Analizar la temática a tratar e identificar sus temas principales (métodos de ordenación burbuja, selección e inserción).

Objetivo específico 2: Tomar las fuentes necesarias para la explicación de cada tema.

Objetivo específico 3: Organizar de manera dinámica la información que se quiere mostrar en una página web utilizando la herramienta de diseño de mockups (Balsamig Mockups).
